Overview

A set of Web Components dealing with Scripture texts (i.e. the bible). See the docs for details.

Scripture Passage

<scripture-passage> puts a passage of scripture on the page. You provide a reference and an api.

Simple use case

<scripture-passage api="crosswayApi" reference="Acts 17:24-25">
</scripture-passage>

Open by default

<scripture-passage
    open
    api="crosswayApi"
    reference="2 Corinthians 10:17-18">
</scripture-passage>

Multiple Passages

<scripture-passage
    open
    api="crosswayApi"
    reference="Genesis 45:5-8, Genesis 50:20">
</scripture-passage>

Scripture List

<scripture-list> allows you to add multiple scripture passages to the page, and expand or collapse all of them together

note: you must use single quotes around the reference-list attribute value, so that you can use proper JSON double quotes in the value 

<scripture-list
    api="crosswayApi"
    reference-list='["2 Timothy 1:9", "James 1:18"]'>
</scripture-list>
